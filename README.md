# Mini Bug Tracker – JavaScript Practice

A small bug tracking interface built with vanilla JavaScript.

## Features

- Bug filtering by status and priority
- Combined filters (important, recurring, etc.)
- Event delegation using event bubbling
- Dynamic DOM rendering
- Active UI filter state

## Concepts Practiced

- DOM manipulation
- Event bubbling
- Event delegation
- Conditional filtering
- UI state management
